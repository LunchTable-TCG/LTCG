# Welcome to LTCG Documentation

**Lunchtable Trading Card Game (LTCG)** is a Yu-Gi-Oh! inspired multiplayer card game built with modern web technologies.

## Tech Stack

- **Frontend:** Next.js 15, React 19, TypeScript, Tailwind CSS 4
- **Backend:** Convex (real-time database + serverless functions)
- **Auth:** Privy + Solana wallet authentication
- **Runtime:** Bun 1.3+
- **Deployment:** Vercel

## Quick Start

### First-Time Setup

Use the interactive setup wizard:

```bash
# 1. Install dependencies
bun install

# 2. Configure environment
cp .env.example .env.local
# Edit .env.local with your Convex and Privy credentials

# 3. Run setup wizard
bun run setup:wizard
```
