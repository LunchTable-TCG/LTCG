# Story Mode Guide

**Story mode** is a single-player campaign where players battle AI opponents across **10 chapters**, each featuring unique archetypes and increasing difficulty.

## Key Features

<Callout type="info">
- ✅ 10 chapters across 2 acts
- ✅ AI opponents with archetype-specific decks
- ✅ Progression tracking and rewards
- ✅ XP and leveling system
- ✅ Badge achievements
</Callout>

<Callout type="warn">
**In Development**: Stage system (10 stages per chapter) is being implemented. Currently, battles start directly from chapter selection.
</Callout>

## How Story Mode Works

### Data Flow Diagram

```
┌─────────────────────────────────────────┐
│   Story Hub (/play/story)              │
│   - Lists all 10 chapters              │
│   - Shows player progress              │
│   - Chapter 1 unlocked by default      │
└──────────────┬──────────────────────────┘
               │ Click Chapter
               ▼
┌─────────────────────────────────────────┐
│   Chapter Detail Page                   │
│   - Shows chapter info and rewards     │
│   - Displays AI opponent deck          │
│   - Start Battle button                │
└──────────────┬──────────────────────────┘
               │ Start Battle
               ▼
┌─────────────────────────────────────────┐
│   initializeStoryBattle                 │
│   - Creates AI opponent                 │
│   - Builds AI deck from archetype       │
│   - Initializes game state              │
│   - Returns lobbyId and gameId          │
└──────────────┬──────────────────────────┘
               │
               ▼
┌─────────────────────────────────────────┐
│   Battle Page                           │
│   - Renders GameBoard component         │
│   - Player takes turn                   │
│   - AI auto-executes turn after player  │
│   - Loops until winner determined       │
└──────────────┬──────────────────────────┘
               │ Battle Ends
               ▼
┌─────────────────────────────────────────┐
│   completeChapter                       │
│   - Calculate star rating (LP-based)    │
│   - Award gold (base + bonus)           │
│   - Award XP (with level tracking)      │
│   - Award card rewards                  │
│   - Unlock next chapter                 │
└─────────────────────────────────────────┘
```

## Backend API Reference

### Queries

<Accordions>
<Accordion title="getPlayerProgress">

```typescript
api.progression.story.getPlayerProgress()
```

**Returns:**
```typescript
{
  totalStarsEarned: number;
  chaptersCompleted: number;
  lastPlayedAt?: string;
}
```

</Accordion>

<Accordion title="getChapterDetails">

```typescript
api.progression.story.getChapterDetails({
  actNumber: 1,
  chapterNumber: 1
})
```

**Returns:** Chapter data including title, description, opponent info, and player progress

</Accordion>

<Accordion title="getAvailableChapters">

```typescript
api.progression.story.getAvailableChapters({
  difficulty: "normal" // or "hard" | "legendary"
})
```

**Returns:** Array of chapters with unlock status and completion stars

</Accordion>

<Accordion title="getPlayerXPInfo">

```typescript
api.progression.story.getPlayerXPInfo()
```

**Returns:**
```typescript
{
  level: number;
  currentXP: number;
  xpToNextLevel: number;
}
```

</Accordion>
</Accordions>

### Mutations

<Accordions>
<Accordion title="initializeStoryBattle">

```typescript
api.progression.story.initializeStoryBattle({
  actNumber: 1,
  chapterNumber: 1,
  difficulty: "normal"
})
```

**Returns:**
```typescript
{
  lobbyId: Id<"gameLobbies">;
  gameId: Id<"gameStates">;
}
```

Navigate to `/play/battle?lobbyId={lobbyId}` to start the battle.

</Accordion>

<Accordion title="completeChapter">

```typescript
api.progression.story.completeChapter({
  chapterNumber: 1,
  starsEarned: 3,
  finalLifePoints: 8000
})
```

**Calculates Rewards:**
- **Gold**: Base 100 + (stars × 50) + (remaining LP × 0.01)
- **XP**: Base 50 + (stars × 25)
- **Cards**: Chance for rare card drops (3 stars = higher chance)

</Accordion>
</Accordions>

## Rewards System

### Star Rating

Stars are awarded based on remaining Life Points (LP) at victory:

| Remaining LP | Stars |
|--------------|-------|
| 5000+ LP | ⭐⭐⭐ (3 stars) |
| 2000-4999 LP | ⭐⭐ (2 stars) |
| 1-1999 LP | ⭐ (1 star) |

### Gold Formula

```typescript
const baseGold = 100;
const starBonus = starsEarned * 50;
const lpBonus = Math.floor(remainingLP * 0.01);
const totalGold = baseGold + starBonus + lpBonus;
```

**Example**: Winning with 6000 LP and 3 stars:
- Base: 100
- Star bonus: 3 × 50 = 150
- LP bonus: 6000 × 0.01 = 60
- **Total**: 310 gold

### XP Formula

```typescript
const baseXP = 50;
const starBonus = starsEarned * 25;
const totalXP = baseXP + starBonus;
```

**Leveling**: Each level requires `level × 100` XP. Max level is currently 50.

## Tips & Strategies

<Callout>
### For New Players

1. **Complete Chapter 1 first** - It teaches basic mechanics
2. **Build a balanced deck** - Mix creatures, spells, and support cards
3. **Watch AI patterns** - Each archetype has predictable strategies
4. **Save gold for packs** - Better cards make later chapters easier
</Callout>

<Callout type="warn">
### For Advanced Players

1. **Perfect 3-star runs** - Maximize rewards by finishing with high LP
2. **Counter AI archetypes** - Build decks specifically for each chapter
3. **Farm favorite chapters** - Replay for gold and XP
4. **Track achievements** - Complete special challenges for badges
</Callout>
