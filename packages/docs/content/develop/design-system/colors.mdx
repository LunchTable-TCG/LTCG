# Color System

LTCG uses a dark fantasy aesthetic with gold accents.

## Color Palette

### Background Colors

Dark slate tones for depth and contrast:

```css
/* Primary backgrounds */
bg-slate-950  /* Darkest - main background */
bg-slate-900  /* Dark - card backgrounds */
bg-slate-800  /* Medium - elevated surfaces */

/* With transparency */
bg-slate-900/50   /* 50% opacity */
bg-slate-800/30   /* 30% opacity */
```

### Accent Colors

Gold/amber tones for highlights and CTAs:

```css
/* Borders */
border-amber-500  /* Primary gold border */

/* Text */
text-amber-400    /* Bright gold text */
text-amber-300    /* Light gold */
text-amber-200    /* Lighter gold */

/* Backgrounds */
bg-amber-500      /* Solid gold background */
```

### Text Colors

Slate grays for readability:

```css
text-slate-300    /* Primary text */
text-slate-400    /* Secondary text */
text-slate-500    /* Tertiary text */
```

### Status Colors

For active indicators and success states:

```css
/* Success/Active */
bg-emerald-500    /* Active indicator background */
text-emerald-400  /* Active indicator text */
bg-green-500      /* Success state */

/* Warning */
bg-yellow-500     /* Warning state */
text-yellow-400   /* Warning text */

/* Error */
bg-red-500        /* Error state */
text-red-400      /* Error text */
```

## Design Tokens

### Gradients

```css
/* Text gradients */
.gradient-gold {
  background: linear-gradient(to right, #fbbf24, #f59e0b);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}

/* Background gradients */
.gradient-dark {
  background: linear-gradient(to bottom, #0f172a, #1e293b);
}
```

```css
/* Glow effects */
.glow-gold {
```

## Component Patterns

### Card Styling

Standard card with dark fantasy aesthetic:

```tsx
<div className="relative p-8 bg-slate-900/50 backdrop-blur-sm border border-slate-700 rounded-xl">
  {/* Corner decorations */}
  <div className="absolute top-0 left-0 w-3 h-3 border-t-2 border-l-2 border-amber-500" />
  <div className="absolute top-0 right-0 w-3 h-3 border-t-2 border-r-2 border-amber-500" />

  {/* Content */}
  <h3 className="text-slate-400 mb-2">Title</h3>
  <p className="text-4xl font-bold bg-gradient-to-r from-amber-400 to-amber-200 bg-clip-text text-transparent">
    Value
  </p>
</div>
```

### Hover States

```css
/* Card hover */
.card-hover {
  transition: all 0.3s ease;
}

.card-hover:hover {
  transform: scale(1.02);
  box-shadow: 0 0 20px rgba(251, 191, 36, 0.3);
  border-color: rgb(251, 191, 36);
}
```

### Active Indicators

```tsx
<div className="flex items-center gap-2">

</div>
```

## Accessibility

Ensure sufficient contrast ratios:

- **Primary text on dark bg**: `text-slate-300` on `bg-slate-950` (14.1:1)
- **Gold accent text**: `text-amber-400` on `bg-slate-900` (7.2:1)
- **Success indicators**: `text-emerald-400` on `bg-slate-900` (5.8:1)

All combinations meet WCAG AAA standards for normal text.
