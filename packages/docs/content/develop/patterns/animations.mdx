# Animation Patterns

Guide to implementing animations in LTCG components.

## Counter Animations

Animated number counters with spring physics for statistics displays.

### Implementation

```typescript
import { useInView, useSpring, useTransform, motion } from "framer-motion";
import { useEffect, useRef } from "react";

function AnimatedCounter({ value, delay = 0 }: { value: number; delay?: number }) {
  const ref = useRef<HTMLSpanElement>(null);
  const isInView = useInView(ref, { once: true });

  const spring = useSpring(0, { damping: 60, stiffness: 100 });
  const display = useTransform(spring, (current) =>
    Math.round(current).toLocaleString()
  );

  useEffect(() => {
    if (isInView) {
      setTimeout(() => {
        spring.set(value);
      }, delay);
    }
  }, [isInView, value, spring, delay]);

  return <motion.span ref={ref}>{display}</motion.span>;
}
```

### Key Features

1. **Viewport Detection**: Uses `useInView` hook to trigger animation when scrolled into view
2. **Spring Physics**: Smooth count-up with `damping: 60, stiffness: 100`
3. **Motion Values**: Tracks current value and updates display reactively
4. **Staggered Timing**: Optional delay for cascading animations

### Number Formatting

Format large numbers with K/M notation:

```typescript
function formatNumber(num: number): string {
  if (num >= 1000000) return `${(num / 1000000).toFixed(1)}M+`;
  if (num >= 1000) return `${(num / 1000).toFixed(1)}K+`;
  return num.toString();
}
```

## Staggered Card Animations

Animate multiple cards with delays for visual appeal:

```typescript
const cards = [
  { title: "Total Players", value: stats.totalPlayers, delay: 0 },
  { title: "Games Today", value: stats.gamesToday, delay: 150 },
  { title: "Total Battles", value: stats.totalBattles, delay: 300 },
];

return (
  <div className="grid grid-cols-1 md:grid-cols-3 gap-8">
    {cards.map((card, index) => (
      <motion.div
        key={card.title}
        initial={{ opacity: 0, y: 20 }}
        animate={{ opacity: 1, y: 0 }}
        transition={{ delay: card.delay / 1000 }}
      >
        <AnimatedCounter value={card.value} delay={card.delay} />
      </motion.div>
    ))}
  </div>
);
```

## Hover Effects

Subtle glow and scale effects on hover:

```typescript
<motion.div
  className="relative p-8 bg-slate-900/50 backdrop-blur-sm border border-slate-700 rounded-xl"
  whileHover={{ scale: 1.02 }}
  transition={{ duration: 0.2 }}
>
  {/* Card content */}
</motion.div>
```

### CSS Hover Effects

Combine with CSS for glow effects:

```css
.card-glow {
  transition: box-shadow 0.3s ease;
}

.card-glow:hover {
  box-shadow: 0 0 20px rgba(251, 191, 36, 0.3);
}
```

## Pulse Animations

For active indicators and attention-grabbing elements:

```typescript
<motion.div
  className="w-2 h-2 bg-emerald-500 rounded-full"
  animate={{ scale: [1, 1.2, 1], opacity: [1, 0.8, 1] }}
  transition={{ duration: 2, repeat: Infinity }}
/>
```

## Loading Skeletons

Animated placeholders while data loads:

```typescript
function StatCardSkeleton() {
  return (
    <div className="p-8 bg-slate-900/50 backdrop-blur-sm border border-slate-700 rounded-xl">
      <motion.div
        className="h-4 bg-slate-700 rounded mb-4"
        animate={{ opacity: [0.5, 1, 0.5] }}
        transition={{ duration: 1.5, repeat: Infinity }}
      />
      <motion.div
        className="h-8 bg-slate-600 rounded"
        animate={{ opacity: [0.5, 1, 0.5] }}
        transition={{ duration: 1.5, repeat: Infinity, delay: 0.2 }}
      />
    </div>
  );
}
```

## Performance Considerations

### Viewport-Based Triggering

Only animate when visible to save resources:

```typescript
const isInView = useInView(ref, { once: true, margin: "-100px" });
```

### Hardware Acceleration

Use transform properties for better performance:

```typescript
// Good - GPU accelerated
transform: "translateY(20px)"

// Avoid - triggers layout reflow
top: "20px"
```

### Reduced Motion

Respect user preferences:

```typescript
import { useReducedMotion } from "framer-motion";

function MyComponent() {
  const shouldReduceMotion = useReducedMotion();

  return (
    <motion.div
      animate={shouldReduceMotion ? {} : { scale: [1, 1.2, 1] }}
    />
  );
}
```
