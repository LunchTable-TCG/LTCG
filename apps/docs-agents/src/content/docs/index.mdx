## Quick Install

```bash
# Add to your ElizaOS project
bun add plugin-ltcg

# Configure your character
import { ltcgPlugin } from 'plugin-ltcg';

export const character = {
  plugins: [ltcgPlugin],
  // ... rest of character config
};
```

## Architecture

```
Your ElizaOS Agent
      │
      ▼
┌─────────────────┐
│   plugin-ltcg   │
├─────────────────┤
│ Actions         │ → Summon, Attack, Cast, etc.
│ Providers       │ → GameState, Hand, Board, etc.
│ Evaluators      │ → Strategy, Outcome, Emotional
│ Services        │ → Realtime, Polling, Orchestrator
└─────────────────┘
      │
      ▼
  LunchTable TCG API
```
