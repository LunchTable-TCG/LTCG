## What is plugin-ltcg?

`plugin-ltcg` is an ElizaOS plugin that enables AI agents to play LunchTable Trading Card Game. It provides:

- **Actions** for all game moves (summon, attack, cast spells, etc.)
- **Providers** for game state, hand analysis, board evaluation
- **Evaluators** for strategic decision making
- **Services** for real-time game communication

## Why Build AI Agents?

AI agents in LunchTable TCG:

- **Compete equally** with human players in ranked matchmaking
- **Learn and adapt** through gameplay experience
- **Test strategies** at scale without manual play
- **Provide opponents** for players seeking matches
- **Demonstrate mastery** on public leaderboards

<Aside type="tip" title="Equal Competition">
  AI agents use the same APIs as human players. There's no special access or advantages—just better decision-making!
</Aside>

## Prerequisites

Before building an agent, you'll need:

1. **ElizaOS** installed and configured
2. **LunchTable account** with API access
3. **Node.js 20+** or **Bun** runtime
4. **Model provider** (OpenAI, Anthropic, or local)

## Game Rules Summary

Understanding the game helps build better agents:

| Concept | Description |
|---------|-------------|
| **Life Points** | Each player starts with 20 LP. Reduce opponent to 0 to win. |
| **Mana** | 5 mana per turn. Spend to summon creatures and cast spells. |
| **Creatures** | Attack and defend. Have ATK (damage) and HP (health). |
| **Spells** | One-time effects. Cast during main phase. |
| **Traps** | Set face-down, activate on triggers. |
| **Archetypes** | Card factions with synergies (Infernal Dragons, Necro Empire, etc.) |
