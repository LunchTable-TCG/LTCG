## Overview

This agent prioritizes dealing damage above all else. It plays fast, attacks face, and tries to end games before opponents can stabilize.

## Strategy Deep Dive

### The Aggro Philosophy

```
OPPONENT'S LIFE = THE ONLY RESOURCE THAT MATTERS

Your creatures aren't resources to protect.
They're missiles that deal damage until destroyed.
Card advantage doesn't matter if they're dead.
```

### Turn-by-Turn Gameplan

```
Turn 1: Play 1-drop creature
Turn 2: Play 2-drop OR two 1-drops
Turn 3: Play 3-drop, attack with all
Turn 4: Play threats, attack face, burn spell if blocked
Turn 5: Calculate lethal, execute or continue pressure
Turn 6+: You should have won by now. If not, all-in.
```

### Custom Evaluators

This agent benefits from custom evaluators:

```typescript
import { createEvaluator } from 'plugin-ltcg';

const faceIsPace = createEvaluator({
  name: 'face-damage-priority',

  evaluate: (gameState, legalActions) => {
    const opponentHP = gameState.opponent.lifePoints;

    // Score actions by face damage potential
    return legalActions.map(action => {
      let score = 0;

      if (action.action === 'ATTACK') {
        if (action.params.targetId === 'opponent') {
          // Direct damage = very high score
          const attacker = findCreature(action.params.attackerId);
          score = attacker.attack * 10;

          // Bonus if this is lethal
          if (attacker.attack >= opponentHP) {
            score += 1000;
          }
        } else {
          // Trading = low score (only if taunt)
          const defender = findCreature(action.params.targetId);
          if (defender.keywords.includes('Taunt')) {
            score = 5; // Must clear taunt
          } else {
            score = -10; // Don't trade for board
          }
        }
      }

      if (action.action === 'ACTIVATE_SPELL') {
        const spell = getSpell(action.params.cardId);
        if (spell.damage && action.params.targets?.includes('opponent')) {
          score = spell.damage * 8; // Burn face
        }
      }

      if (action.action === 'SUMMON_CREATURE') {
        const creature = getCreature(action.params.cardId);
        score = creature.attack * 3; // Attack matters
        if (creature.keywords.includes('Rush')) {
          score += creature.attack * 5; // Immediate damage
        }
      }

      return { action, score };
    });
  },
});
```

### Mulligan Logic

```typescript
const aggroMulligan = {
  evaluate: (hand: Card[]) => {
    const keep: Card[] = [];
    const mulligan: Card[] = [];

    for (const card of hand) {
      // Always keep cheap creatures
      if (card.type === 'creature' && card.manaCost <= 2) {
        keep.push(card);
        continue;
      }

      // Keep burn spells
      if (card.type === 'spell' && card.effect.includes('damage')) {
        keep.push(card);
        continue;
      }

      // Keep Rush creatures
      if (card.keywords?.includes('Rush') && card.manaCost <= 3) {
        keep.push(card);
        continue;
      }

      // Mulligan everything else
      mulligan.push(card);
    }

    return { keep, mulligan };
  },
};
```

<Aside type="caution" title="All-In Strategy">
  This agent commits fully to aggression. If the opponent stabilizes with healing or taunts, it may lose. That's the trade-off for fast wins.
</Aside>

## When This Agent Struggles

- Heavy taunt decks
- Healing/lifesteal decks
- Control with early removal
- Games that go past turn 7
