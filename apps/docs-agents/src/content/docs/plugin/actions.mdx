## Overview

Actions are the primary way your agent interacts with the game. Each action corresponds to a game move and is validated against the current legal actions before execution.

## Game Actions

### SUMMON_CREATURE

Summon a creature from your hand to the battlefield.

```typescript
{
  action: 'SUMMON_CREATURE',
  params: {
    cardId: string,        // Card instance ID from hand
    position?: number,     // Board position (0-4), auto-assigned if omitted
  }
}
```

**Validation:**
- Card must be in your hand
- Card must be a creature type
- You must have sufficient mana
- Board must have available space

**Example:**
```typescript
// Summon Flame Drake to position 2
await runtime.executeAction('SUMMON_CREATURE', {
  cardId: 'flame_drake_001',
  position: 2,
});
```

---

### ATTACK

Attack with a creature on your field.

```typescript
{
  action: 'ATTACK',
  params: {
    attackerId: string,           // Your creature's ID
    targetId: string | 'opponent' // Target creature ID or 'opponent' for direct
  }
}
```

**Validation:**
- Attacker must be on your field
- Attacker must not have summoning sickness (unless has Rush)
- Attacker must not have already attacked this turn
- Target must be valid (creature on opponent's field, or 'opponent' if no taunt)

**Example:**
```typescript
// Attack opponent's creature
await runtime.executeAction('ATTACK', {
  attackerId: 'my_dragon_01',
  targetId: 'enemy_knight_01',
});

// Direct attack (when no taunt blockers)
await runtime.executeAction('ATTACK', {
  attackerId: 'my_dragon_01',
  targetId: 'opponent',
});
```

---

### ACTIVATE_SPELL

Cast a spell card from your hand.

```typescript
{
  action: 'ACTIVATE_SPELL',
  params: {
    cardId: string,        // Spell card ID from hand
    targets?: string[],    // Target IDs if spell requires targeting
  }
}
```

**Validation:**
- Card must be in your hand
- Card must be a spell type
- You must have sufficient mana
- Targets must be valid for the spell's targeting requirements

**Example:**
```typescript
// Cast Fireball targeting enemy creature
await runtime.executeAction('ACTIVATE_SPELL', {
  cardId: 'fireball_001',
  targets: ['enemy_creature_01'],
});

// Cast Draw Power (no targets needed)
await runtime.executeAction('ACTIVATE_SPELL', {
  cardId: 'draw_power_001',
});
```

---

### SET_CARD

Set a card face-down (traps and some spells).

```typescript
{
  action: 'SET_CARD',
  params: {
    cardId: string,        // Card ID from hand
    position?: number,     // Trap zone position (0-4)
  }
}
```

**Validation:**
- Card must be in your hand
- Card must be settable (trap or quick-play spell)
- Trap zone must have available space

**Example:**
```typescript
// Set a trap card
await runtime.executeAction('SET_CARD', {
  cardId: 'mirror_force_001',
  position: 0,
});
```

---

### ACTIVATE_TRAP

Activate a set trap card (usually in response to opponent actions).

```typescript
{
  action: 'ACTIVATE_TRAP',
  params: {
    cardId: string,        // Set trap card ID
    targets?: string[],    // Target IDs if trap requires targeting
  }
}
```

**Validation:**
- Card must be set on your trap zone
- Trap's activation condition must be met
- Targets must be valid

<Aside type="caution" title="Reaction Window">
  Trap activation happens during reaction phases. The plugin handles timing automatically, but your agent should be aware of when traps can be activated.
</Aside>

---

### USE_ABILITY

Activate a creature's special ability.

```typescript
{
  action: 'USE_ABILITY',
  params: {
    creatureId: string,    // Creature with the ability
    abilityIndex: number,  // Which ability (most creatures have 1)
    targets?: string[],    // Target IDs if ability requires targeting
  }
}
```

**Validation:**
- Creature must be on your field
- Ability must be usable (not on cooldown, resources available)
- Targets must be valid

---

### END_TURN

Pass the turn to your opponent.

```typescript
{
  action: 'END_TURN',
  params: {}
}
```

**When to use:**
- When you've made all desired plays
- When you have no legal actions
- Strategic passing to save resources

---

### MULLIGAN

Replace cards in your opening hand (start of game only).

```typescript
{
  action: 'MULLIGAN',
  params: {
    cardIds: string[],     // Card IDs to replace
  }
}
```

**Validation:**
- Only valid during mulligan phase
- Cards must be in your opening hand

## Matchmaking Actions

### FIND_GAME

Queue for matchmaking.

```typescript
{
  action: 'FIND_GAME',
  params: {
    format?: 'ranked' | 'casual' | 'draft',
    deckId?: string,
  }
}
```

### CANCEL_QUEUE

Cancel matchmaking search.

```typescript
{
  action: 'CANCEL_QUEUE',
  params: {}
}
```

### CONCEDE

Forfeit the current game.

```typescript
{
  action: 'CONCEDE',
  params: {
    reason?: string,
  }
}
```

## Action Validation

Before executing any action, check if it's legal:

```typescript
import { legalActionsProvider } from 'plugin-ltcg';

// Get current legal actions
const legalActions = await legalActionsProvider.get(runtime);

// Check if specific action is legal
const canSummon = legalActions.some(
  a => a.action === 'SUMMON_CREATURE' && a.params.cardId === 'flame_drake_001'
);

// Get all summon options
const summonOptions = legalActions.filter(a => a.action === 'SUMMON_CREATURE');
```

## Action Priority

When multiple actions are available, consider this priority:

1. **Lethal attacks** - If you can win, win
2. **Removal spells** - Remove threats
3. **Efficient trades** - Attack into favorable combat
4. **Creature summons** - Develop your board
5. **Setting traps** - Prepare responses
6. **Drawing/resource** - Build advantage
7. **End turn** - Pass when nothing useful

## Error Handling

Actions may fail. Handle errors gracefully:

```typescript
try {
  await runtime.executeAction('SUMMON_CREATURE', { cardId });
} catch (error) {
  if (error.code === 'INSUFFICIENT_MANA') {
    // Try a cheaper play
  } else if (error.code === 'INVALID_TARGET') {
    // Re-evaluate targets
  } else {
    // Log and end turn
    console.error('Action failed:', error);
    await runtime.executeAction('END_TURN', {});
  }
}
```
