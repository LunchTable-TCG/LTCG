## Overview

LunchTable TCG uses a dark fantasy color palette inspired by weathered leather, aged parchment, and mystical elements.

## Core Palette

### Background Colors

| Token | Value | Usage |
|-------|-------|-------|
| `--ltcg-bg-primary` | `#1a1614` | Main background |
| `--ltcg-bg-secondary` | `#130f0d` | Navigation, sidebars |
| `--ltcg-bg-elevated` | `#252019` | Cards, modals |
| `--ltcg-bg-hover` | `#2d2620` | Hover states |

### Text Colors

| Token | Value | Usage |
|-------|-------|-------|
| `--ltcg-text-primary` | `#e8e0d5` | Primary text |
| `--ltcg-text-secondary` | `#b5a99a` | Secondary text |
| `--ltcg-text-muted` | `#7a6f63` | Muted text |
| `--ltcg-text-accent` | `#d4af37` | Gold accent text |

### Accent Colors

| Token | Value | Usage |
|-------|-------|-------|
| `--ltcg-gold` | `#d4af37` | Primary accent |
| `--ltcg-gold-light` | `#f4d03f` | Hover state |
| `--ltcg-gold-dark` | `#b8960c` | Active state |

| Element | Primary | Glow | Usage |
|---------|---------|------|-------|
| Ember | `#ff6b35` | `rgba(255, 107, 53, 0.3)` | Fire, aggression |
| Void | `#9333ea` | `rgba(147, 51, 234, 0.3)` | Dark, disruption |
| Arcane | `#06b6d4` | `rgba(6, 182, 212, 0.3)` | Magic, control |
| Nature | `#10b981` | `rgba(16, 185, 129, 0.3)` | Growth, healing |
| Frost | `#0ea5e9` | `rgba(14, 165, 233, 0.3)` | Ice, defense |
| Neutral | `#a89f94` | `rgba(168, 159, 148, 0.3)` | Colorless |

|-------|-------|-------|
| Your Turn | `#22c55e` | Active player indicator |

## Usage Guidelines

### Do

- Use element colors consistently for card borders and badges
- Use gold accent sparingly for emphasis
- Maintain sufficient contrast (4.5:1 minimum)
- Use semantic colors for their intended purpose

### Don't

- Mix element colors arbitrarily
- Use bright colors on dark backgrounds without glow
- Override semantic color meanings
- Use pure white (`#fff`) - prefer parchment tones

## Applying Colors

### In CSS

```css
.card {
  background: var(--ltcg-bg-elevated);
  border: 1px solid var(--ltcg-gold-dark);
  color: var(--ltcg-text-primary);
}

.card:hover {
  border-color: var(--ltcg-gold);
  box-shadow: 0 0 20px var(--ltcg-gold-glow);
}
```

### In Tailwind

```typescript
// tailwind.config.ts
export default {
  theme: {
    extend: {
      colors: {
        ltcg: {
          bg: {
            primary: "#1a1614",
            secondary: "#130f0d",
            elevated: "#252019",
          },
          ember: "#ff6b35",
          void: "#9333ea",
          // ... etc
        },
      },
    },
  },
};
```

```jsx
<div className="bg-ltcg-bg-primary text-ltcg-text-primary border-ltcg-gold">
  Content
</div>
```

## Accessibility

<Aside type="tip">
  All color combinations meet WCAG 2.1 AA standards for contrast.
</Aside>

| Combination | Contrast Ratio | Rating |
|-------------|----------------|--------|
| Primary text on primary bg | 12.5:1 | AAA |
| Secondary text on primary bg | 7.2:1 | AAA |
| Gold on primary bg | 8.1:1 | AAA |
| Muted text on primary bg | 4.6:1 | AA |
