## Overview

LunchTable TCG uses a fantasy-inspired typography system that balances readability with thematic atmosphere.

## Font Families

### Cinzel (Headings)

A classically-inspired serif typeface with sharp serifs and tall proportions.

```css
font-family: "Cinzel", serif;
```

**Usage:**
- Page titles
- Section headings
- Card names
- UI headers

### Crimson Text (Body)

A readable serif designed for long-form text, with old-style numerals.

```css
font-family: "Crimson Text", serif;
```

**Usage:**
- Body text
- Card descriptions
- UI labels
- Navigation

### JetBrains Mono (Code)

A monospace font optimized for code readability.

```css
font-family: "JetBrains Mono", monospace;
```

**Usage:**
- Code blocks
- Technical values
- Stats display

## CSS Variables

```css
:root {
  /* Font families */
  --ltcg-font-heading: "Cinzel", serif;
  --ltcg-font-body: "Crimson Text", serif;
  --ltcg-font-mono: "JetBrains Mono", monospace;

  /* Font sizes */
  --ltcg-text-xs: 0.75rem;
  --ltcg-text-sm: 0.875rem;
  --ltcg-text-base: 1rem;
  --ltcg-text-lg: 1.25rem;
  --ltcg-text-xl: 1.5rem;
  --ltcg-text-2xl: 1.875rem;
  --ltcg-text-3xl: 2.25rem;
  --ltcg-text-4xl: 3rem;

  /* Font weights */
  --ltcg-font-normal: 400;
  --ltcg-font-medium: 500;
  --ltcg-font-semibold: 600;
  --ltcg-font-bold: 700;

  /* Line heights */
  --ltcg-leading-tight: 1.1;
  --ltcg-leading-snug: 1.25;
  --ltcg-leading-normal: 1.5;
  --ltcg-leading-relaxed: 1.6;
}
```

## Text Styles

### Headings

```css
h1, .h1 {
  font-family: var(--ltcg-font-heading);
  font-size: var(--ltcg-text-3xl);
  font-weight: var(--ltcg-font-semibold);
  line-height: var(--ltcg-leading-tight);
  color: var(--ltcg-text-primary);
  letter-spacing: 0.02em;
}

h2, .h2 {
  font-family: var(--ltcg-font-heading);
  font-size: var(--ltcg-text-2xl);
  font-weight: var(--ltcg-font-semibold);
  line-height: var(--ltcg-leading-snug);
  color: var(--ltcg-text-primary);
}
```

### Body Text

```css
body {
  font-family: var(--ltcg-font-body);
  font-size: var(--ltcg-text-base);
  font-weight: var(--ltcg-font-normal);
  line-height: var(--ltcg-leading-relaxed);
  color: var(--ltcg-text-primary);
}

p {
  margin-bottom: 1em;
}
```

### Card Text

```css
.card-name {
  font-family: var(--ltcg-font-heading);
  font-size: var(--ltcg-text-lg);
  font-weight: var(--ltcg-font-semibold);
  text-transform: uppercase;
  letter-spacing: 0.05em;
}

.card-description {
  font-family: var(--ltcg-font-body);
  font-size: var(--ltcg-text-sm);
  font-style: italic;
  line-height: var(--ltcg-leading-normal);
}

.card-stats {
  font-family: var(--ltcg-font-mono);
  font-size: var(--ltcg-text-sm);
  font-weight: var(--ltcg-font-medium);
}
```

## Special Effects

### Gold Accent Text

```css
.text-gold {
  color: var(--ltcg-gold);
  text-shadow: 0 0 10px var(--ltcg-gold-glow);
}
```

### Embossed Effect

```css
.text-embossed {
  color: var(--ltcg-text-primary);
  text-shadow:
    1px 1px 0 var(--ltcg-bg-primary),
    -1px -1px 0 rgba(255, 255, 255, 0.1);
}
```

### Glowing Text

```css
.text-glow {
  animation: text-glow 2s ease-in-out infinite alternate;
}

@keyframes text-glow {
  from {
    text-shadow: 0 0 5px var(--ltcg-gold-glow);
  }
  to {
    text-shadow: 0 0 20px var(--ltcg-gold-glow), 0 0 30px var(--ltcg-gold-glow);
  }
}
```

## Best Practices

### Do

- Use Cinzel for headings and important UI elements
- Use Crimson Text for readable body content
- Maintain hierarchy through size and weight
- Add letter-spacing to uppercase text

### Don't

- Mix more than 3 fonts
- Use thin weights on dark backgrounds
- Apply text shadows to body text
- Use all-caps for long text

<Aside type="tip">
  For card text, use slightly larger sizes than typical UI text to account for the ornate styling.
</Aside>
